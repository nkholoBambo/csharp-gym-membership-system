# csharp-gym-membership-system
Simple system that prints prices for specific age groups for gym memberships

##Features
-Age-based membership pricing
-Custom messages for different age groups
-User input handling
-Conditional logic using if statements

##Age Categories
-Under 5 not eligible for membership
-5 to 12 pay a cheaper price
-13 to 17 pay a better price
-18 to 59 pay the normal price
-60 to 69 pay senior discount
-70+ not eligible for membership
